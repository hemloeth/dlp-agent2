# DLP Scanner Landing Page

This is a modern, high-performance landing page for the DLP Scanner desktop application, built with Next.js 14.

## Features

- **Modern UI**: Dark theme with glassmorphism, gradients, and subtle animations.
- **Direct Download**: Seamlessly serve the `DLPScanner.exe` build.
- **Responsive**: Fully responsive design for all screen sizes.
- **SEO Optimized**: Pre-configured meta tags and semantic HTML.

## Getting Started

1.  **Installation**:
    ```bash
    npm install
    ```

2.  **Development**:
    ```bash
    npm run dev
    ```

3.  **Production Build**:
    ```bash
    npm run build
    ```

## Folder Structure

- `src/app`: Next.js App Router files (pages, layouts, globals styles).
- `public`: Static assets including the executable build.
- `src/components`: Reusable UI components.
- `src/styles`: Custom CSS modules and utility styles.
